文字版坦克大战 - 商鞅 vs 马
游戏概述
这是一款基于 Pygame 开发的 2D 横版闯关游戏，玩家扮演商鞅进行闯关。游戏融合了平台跳跃、解谜和动作射击元素，具有独特的身体部位分离攻击机制。
核心特性
独特的攻击系统：玩家可以分离身体部位（手、腿）作为武器攻击敌人
物理互动机制：真实的碰撞检测和物理模拟
关卡解谜系统：包含开关、门、陷阱等互动元素
角色成长系统：击败敌人获得经验值升级，解锁新能力
多关卡设计：2个精心设计的关卡，难度逐步增加
运行要求
Python 3.7+
Pygame 2.0+
安装与运行
克隆或下载游戏源代码
安装依赖库：
pip install pygame
运行游戏：
python main.py
操作说明
主菜单界面
数字键 1-5：选择关卡
空格键：开始游戏
ESC键：退出游戏
游戏进行中
A/D键：左右移动
W键：普通跳跃
J键：执行当前选中动作
Q键：切换选择身体部位
R键：回收所有身体部位
ESC键：返回主菜单
鼠标点击：发射子弹（如果鼠标控制功能已启用）
游戏结束/胜利界面
R键：重新开始当前关卡
N键：下一关（仅胜利状态且有关卡可用）
ESC键：返回主菜单
游戏机制
身体部位系统
左手/右手：左右发射
头：向上发射
开关与门系统
子弹可以击中开关，控制对应颜色门的开关状态
门在关闭时阻挡所有物体通过
门在开启时可以自由通过
升级系统
击败敌人获得经验值
升级时解锁新的攻击方式
提升最大生命值
游戏状态
主菜单：选择关卡和开始游戏
游戏进行中：控制角色游戏
游戏结束：玩家生命值归零
胜利：完成当前关卡目标
